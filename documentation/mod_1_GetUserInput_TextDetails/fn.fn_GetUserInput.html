Expand description

A module for handling user input to select a Hebrew Bible codex and specific text.

MODULE.FUNCTION() #1 - GET NAME OF CODEX; GET NUMBER OF TEXT(S) CHOSEN; RETURNS -> TUPLE OF (STRING, INTEGER)

This module provides functionality to prompt the user to choose a Hebrew Bible codex and a specific text within that codex, returning the codex name and text number. The user selects a specific text within a chosen Hebrew Bible codex and returns the codex name and text number.

This function takes a codex number, displays a menu of available texts for that codex (e.g., Genesis, Exodus, etc. or Torah for Koren; additional books for Leningrad and MAM), reads the user’s input as a string, converts it to an unsigned 32-bit integer, and returns a tuple containing the codex name and the selected text number. It includes error handling for invalid input.

§Arguments

  • NumberOfCodex - A u32 representing the selected codex (0: None, 1: Koren, 2: Leningrad, 3: MAM).

§Returns

A tuple (String, u32) containing:

  • The name of the codex (e.g., “Codex: 1 - Koren”) as a String.
  • The number of the selected text as a u32 (e.g., 1 for Genesis, 40 for Torah, 0 for Quit).

§Panics

Panics if the input cannot be parsed as a u32 (e.g., non-numeric input). The function uses expect to handle this, displaying “Please enter a valid number”.
